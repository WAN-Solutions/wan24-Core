Class SynchronizedStream<T>

Synchronized stream (synchronizes reading/writing/seeking operations; the base stream should implement each single asynchronous reading/writing method! Any asynchronous reading/writing method which adopts or calls to another asynchronous reading/writing or any seeking method will cause a dead-lock!)
